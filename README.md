*Binary Search Tree Sprint — Spring Boot Project*

A full-stack Spring Boot application that:

Accepts user input of numbers

Builds a Binary Search Tree (BST)

Returns the tree as a JSON representation

Stores all results in an H2 in-memory database

Displays previously generated trees

Includes unit tests, MVC structure, DTOs, repository layer, and clean UI styling

*How to Run the Project*
1) cd bst-sprint
2) ./mvnw spring-boot:run
3) http://localhost:8080/enter-numbers
*Testing*
./mvnw test
